Descrição

Nosso projeto vem com uma proposta de auxílio econômico onde pretendemos oferecer a partir de uma aplicação web cupons de compra a fim de auxiliar tanto os clientes que passam por um momento de dificuldade e recuo financeiro, quanto o mercado que terá uma nova oportunidade de divulgação e de aquecimento de mercado. Neste projeto pretendemos disponibilizar um espaço para cadastramento tanto de clientes para recebimento de todas as ofertas e oportunidades quanto de cadastramento para as lojas que irão fornecer em um único lugar todas as suas promoções. O aplicativo poderá fornecer tanto ofertas de mercadorias quanto de serviços.
